Passpack Importer

Export your Passpack database to an CSV file and then upload it to select which secrets you wish to upload to Mitro. This can be done by logging into Passpack, selecting 'tools', and then export. Select "Comma Separated Values" and then save the output to a CSV file. Be sure to NOT change the order in which password data is displayed.
